Ссылка на страницу датасета: *"https://www.kaggle.com/datasets/blastchar/telco-customer-churn"*

Бизнес-цель проекта: Реализовать автоматизированную технологию анализа поведения клиентов телекоммуникационной компании, которая позволит бизнесу динамически
                     регулировать свою политику по отношению к реалиям рынка.

Структура проекта:
  - Загрузка данных
  - ETL, EDA
  - Models Analysis
  - Создание программного кода на основе ООП
  - Внедрение модели в пользовательский интерфейс (Flask and Tkinter)

Results:
  Precision = 0.52
  Recall = 0.79
  ROC AUC = 0.85
  PR AUC = 0.68
